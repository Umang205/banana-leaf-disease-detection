# Banana Leaf Disease Detection using CNN

This project uses Convolutional Neural Networks (CNN) and TensorFlow to detect banana leaf diseases from images.

## Technologies Used
- Python
- TensorFlow
- Keras
- CNN
- NumPy
- Matplotlib
- Google Colab

## Features
- Banana leaf disease classification
- Deep learning image prediction
- CNN model training
- Dataset preprocessing

## Workflow
1. Dataset extraction
2. Image preprocessing
3. CNN model building
4. Model training
5. Disease prediction

## Dataset Classes
- Healthy
- Black Sigatoka
- Panama Disease
