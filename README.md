# Banana Leaf Disease Detection using CNN

This project uses Convolutional Neural Networks (CNN) and TensorFlow to detect banana leaf diseases from images.

---

## Project Overview

Banana plants are affected by diseases that reduce crop quality and production.  
This project uses Deep Learning and Computer Vision techniques to classify banana leaf diseases automatically from images.

---

## Technologies Used

- Python
- TensorFlow
- Keras
- CNN (Convolutional Neural Network)
- NumPy
- Matplotlib
- Google Colab

---

## Features

- Banana leaf disease classification
- CNN based image prediction
- Deep learning model training
- Dataset preprocessing
- Image augmentation
